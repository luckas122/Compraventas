1. Inicio de la aplicacion

  1. Abre la aplicacion haciendo doble clic en "Tu local 2025" en el escritorio o desde el menu de inicio.
  2. Selecciona la sucursal donde estas trabajando (Sarmiento, Salta, etc.).
  3. La aplicacion se abre en la pestana de Ventas, lista para operar.
Consejo: En Configuracion puedes elegir una sucursal por defecto para no tener que seleccionarla cada vez.

2. Ventas

La pestana de Ventas es donde se realizan las operaciones diarias. Se accede con F3.

Buscar y agregar productos

  1. Escribe el codigo de barras o el nombre del producto en el buscador.
  2. Presiona Enter o haz clic en el boton + para agregarlo a la cesta.
  3. Si usas un lector de codigos de barras, el producto se agrega automaticamente.
Producto no encontrado: Si el producto no existe, la app te preguntara si queres agregarlo como nuevo. Completa codigo, nombre y precio, y se creara al instante.

Borradores

Si necesitas pausar una venta para atender a otro cliente:

  • Guardar borrador (G): Guarda la cesta actual para despues.
  • Cargar borrador (B): Recupera una cesta guardada anteriormente.

Otras funciones

  • Consultar precio (P): Busca el precio de un producto sin agregarlo a la cesta.
  • Devolucion (D): Procesa una devolucion de una venta anterior.
  • Reimprimir ticket (F): Vuelve a imprimir el ultimo ticket o uno seleccionado.
  • WhatsApp (W): Envia el ticket por WhatsApp Web al cliente.
  • PDF: Exporta el ticket como archivo PDF.

3. Finalizar una venta

Cuando la cesta tiene todos los productos, presiona V o el boton de ticket para finalizar.

Paso 1: Elegir metodo de pago

Aparece un popup con dos opciones: Efectivo o Tarjeta.

Pago en Efectivo

  1. Se abre el dialogo de pago en efectivo con el total a pagar.
  2. Ingresa el importe abonado por el cliente (se auto-selecciona).
  3. Si aplica, agrega un descuento en porcentaje (%) o monto fijo ($).
  4. El vuelto se calcula automaticamente.
  5. Opcionalmente, marca "Emitir factura AFIP" si esta habilitado.
  6. Presiona Aceptar para completar la venta e imprimir el ticket.

Pago con Tarjeta

  1. Se abre el dialogo de pago con tarjeta.
  2. Selecciona el numero de cuotas (1 a 12).
  3. Si aplica, ingresa el interes por cuotas (%).
  4. Si aplica, agrega un descuento (% o $).
  5. Selecciona el tipo de comprobante AFIP (si esta habilitado).
  6. Ingresa el CUIT del cliente si es necesario.
  7. Presiona Aceptar para completar la venta.
Navegacion rapida: Usa Enter para saltar entre campos y Tab para avanzar al siguiente.

Ventas del dia

En la parte inferior de la pestana de Ventas se muestra la tabla de ventas del dia con todos los tickets generados, detallando hora, sucursal, total, metodo de pago y mas.

4. Productos

Accede a la pestana de Productos con F1. Aqui gestionas todo el catalogo.

Agregar un producto

  1. Completa los campos: Codigo, Nombre, Precio y opcionalmente Categoria.
  2. Presiona Agregar o el atajo A.
  3. El producto queda registrado y se sincroniza con otras sucursales.

Editar un producto

  • Doble clic sobre una celda en la tabla para editarla directamente.
  • Atajo E: Pide el codigo y abre un editor.

Edicion masiva de precios

  1. Selecciona los productos que queres modificar (casillas de verificacion).
  2. Haz clic en Editar precios masivos.
  3. Elige ajustar por porcentaje (ej: +10%) o por monto fijo (ej: +$500).
  4. Confirma y todos los precios seleccionados se actualizan.

Importar / Exportar

  • Importar desde Excel: Carga productos desde un archivo .xlsx con columnas: codigo, nombre, precio, categoria.
  • Exportar a Excel: Descarga todo el catalogo a un archivo .xlsx.

Imprimir codigos de barra

Selecciona productos y presiona I para imprimir etiquetas con el codigo de barras.

5. Proveedores

Accede con F2. Aqui gestionas la informacion de tus proveedores.

  • Agregar proveedor: Nombre, telefono, numero de cuenta, CBU.
  • Editar: Doble clic en la tabla para modificar datos.
  • Eliminar: Selecciona y elimina proveedores.
  • Registrar pago: Desde la pestana de Ventas, boton "Pagos" para registrar pagos a proveedores.

6. Historial y reportes

Accede con F4. Aqui ves estadisticas y reportes de ventas.

  • KPIs: Total de ventas, productos mas vendidos, comparacion por sucursal.
  • Grafico de ventas diarias: Tendencia visual de las ventas.
  • Ranking de productos: Los productos mas vendidos por cantidad o ingreso.
  • Filtros: Filtra por rango de fechas, sucursal o forma de pago.
  • Exportar a Excel: Descarga los datos filtrados.

7. Configuracion

Accede con F5. Las opciones mas importantes son:

Datos del negocio

Nombre, CUIT, tipo de IVA, telefono y sucursales.

Impresora

Selecciona la impresora de tickets (termica) y la impresora de codigos de barras.

Tickets

Personaliza las plantillas de tickets. Hay 4 plantillas predefinidas (efectivo clasica, efectivo minimalista, tarjeta clasica, tarjeta minimalista). Podes modificarlas o crear las tuyas.

Apariencia

Activa/desactiva el modo oscuro, elige la variante (suave, medio, alto contraste) y personaliza colores.

AFIP / Facturacion electronica

Configura la integracion con AFIP para emitir facturas electronicas:

  • Habilitar/deshabilitar facturacion.
  • Modo: prueba o produccion.
  • CUIT del negocio y punto de venta.
  • Configuracion del SDK de AFIP (API key, URLs).
Importante: Antes de usar en produccion, probalo en modo "test" para verificar que todo funcione correctamente.

Sincronizacion entre sucursales

Permite sincronizar productos, ventas y proveedores entre sucursales usando Firebase:

  • Ingresa la URL de Firebase y el token de autenticacion.
  • Activa la sincronizacion automatica (cada X minutos).
  • Elige que sincronizar: productos, proveedores.
  • Las ventas se sincronizan siempre automaticamente.

Backups

Configura copias de seguridad automaticas diarias y semanales. Podes hacer backup manual en cualquier momento y restaurar desde una copia anterior.

Atajos de teclado

Personaliza todas las teclas de acceso rapido a tu gusto.

8. Dashboard web (acceso remoto)

El dashboard web te permite ver las ventas y editar precios desde cualquier dispositivo (celular, tablet, otra PC).

Como acceder

  1. Abre el archivo dashboard/index.html en un navegador.
  2. Ingresa la URL de Firebase y el token (los mismos de Configuracion > Sincronizacion).
  3. Haz clic en Conectar.

Pestana Ventas

Ve las ventas del dia en tiempo real: total, efectivo vs tarjeta, cantidad de ventas, detalle por sucursal. Se actualiza automaticamente cada 30 segundos.

Pestana Precios

  1. Haz clic en la pestana Precios.
  2. Busca un producto por codigo o nombre.
  3. Haz clic en el boton de editar (lapiz) del producto.
  4. Ingresa el nuevo precio y haz clic en Guardar.
  5. El cambio se sincroniza automaticamente con todas las sucursales.
Consejo: Los cambios de precios desde el dashboard se aplican en las sucursales en el proximo ciclo de sincronizacion (cada 5 minutos por defecto, o al sincronizar manualmente).

10. Consejos utiles

Lector de codigos: Al escanear un producto, se agrega automaticamente a la cesta sin necesidad de presionar Enter.
Sincronizacion: Si no tenes internet en el momento, los cambios se guardan en una cola y se envian automaticamente cuando se recupera la conexion.
Backups: Configura backups automaticos diarios para no perder datos. Podes restaurar en cualquier momento desde Configuracion.
Producto nuevo al vuelo: Si durante una venta buscas un producto que no existe, la app te ofrece crearlo en el momento sin salir de la seccion de ventas.
Precios desde el celular: Usa el Dashboard web para actualizar precios desde cualquier dispositivo con internet. Los cambios llegan a todas las sucursales.
Atajos personalizables: Todos los atajos de teclado se pueden cambiar desde Configuracion > Acceso Rapido.
